Important Ques. on C

1. Who discovered C?

Ans-: Dennis Ritchie

2. What are the basic Datatypes supported in C Programming Language?

Ans-: The datatypes in C language are classified into 2 categories-:

  • Basic datatype
  • Derived datatype
  • Enumerated datatype
  • Void datatype

3. What do you mean by the Scope of the variable?

Ans-: The scope of a variable is an particular block in which you can access that variable.

4. Differentiate between calloc() and malloc()?

Ans-: calloc() and malloc() are dynamic memory allocating functions. malloc() is a function that creates one block of memory of a fixed size and on the other side calloc() is a function that assigns a specified number of blocks of memory to a single variable.

5. What are the valid places where the programmer can apply Break Control Statement?

Ans-: Break control statement is valid to be used inside a loop and switch control statement.

6. Differentiate between Actual Parameters and Formal Parameters.

Ans-: The parameters which are sent during the function calling is known as Actual parameters and the parameters which we received during function defining is known as Formal parameters.

7. Can a C program be compiled or executed in the absence of a main()?

Ans-: The program will be compiled successfully but will not be executed. To execute any C program main() is required.

8. What is a C Token?

Ans-: Keywords, Constants, special Symbols, Strings, Operators, Identifiers used in C program are referred as C Tokens.

9. Why is C called the Mother of all Languages?

Ans-: C introduced many core concepts and data structures like arrays, lists, functions, strings etc. Many languages designed after C are designed on the basis of C language. Hence, it is considered as the mother of all languages.

10. What is the purpose of printf() and scanf() functions in C program?

Ans-: printf() is used to print the values on the screen and scanf() is used to input the values. We need an appropriate datatype format specifier for both printing ans scanning purposes. For Example,

  • %d:
      It is a datatype format specifier used to print and scan an integer value.

  • %s:
      It is a datatype format specifier used to print and scan an string.

  • %c:
      It is a datatype format specifier used to print and scan an character.

  • %f:
      It is a datatype format specifier used to print and scan an floating value.
